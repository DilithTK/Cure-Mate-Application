# 💊 CureMate - Healthcare & Pharmacy System

CureMate is a Flutter-based mobile application that connects patients with nearby pharmacies for faster prescription handling and healthcare support.

---

## 🚀 Features
- 📄 Upload prescriptions
- 🏥 Find nearby pharmacies
- 💬 Pharmacy response system
- ⏰ Medicine reminders
- 🤖 AI health assistant
- 👤 User authentication (Firebase)

---

## 🛠️ Tech Stack
- Flutter (Dart)
- Firebase (Auth, Firestore, Storage)
- Node.js (Cloud Functions)
- Android / iOS / Web support

---

## 📱 App Modules
- Authentication (Login/Signup)
- User Dashboard
- Pharmacy Dashboard
- Prescription Upload System
- Profile Management

---

## 🔐 Security Note
Sensitive keys (API keys, Firebase config) are not exposed in this repository.
